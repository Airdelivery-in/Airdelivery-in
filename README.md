## ✈️ Our Aircraft

A purpose-built hybrid VTOL drone designed for **practical, scalable delivery operations**.

### Flight Capabilities

| Capability | Description |
|------------|-------------|
| **Hybrid HTOL/VTOL** | Horizontal takeoff for efficient transit; vertical landing for precision delivery in tight spaces |
| **Vector Thrust System** | Advanced vectored thrust enables true VTOL maneuverability and landing accuracy |
| **Fully Autonomous** | End-to-end autonomous flight planning, navigation, obstacle avoidance, and delivery management |

### Technical Specifications

| Specification | Value |
|---------------|-------|
| **Payload Capacity** | 2 kg |
| **Operating Range** | 30 km |
| **Takeoff Modes** | HTOL (efficient cruise) & VTOL (precision landing) |
| **Thrust System** | Vector Thrust |
| **Operation** | Fully Autonomous — no pilot required |

## 🚀 Why AirDelivery?

### Built for Real Operations

- **Efficiency First** — HTOL cruising extends range and reduces energy consumption on long routes
- **Precision Landing** — VTOL + vector thrust enables accurate delivery to backyards, rooftops, and remote sites
- **Autonomous by Design** — No remote pilot needed; intelligent systems handle flight, navigation, and delivery
- **Scalable Infrastructure** — Designed for distributed networks, not one-off demos

### Target Use Cases

- 🏥 **Medical & pharmaceutical** — urgent supplies, lab samples, temperature-sensitive meds
- 📦 **E-commerce last-mile** — same-day delivery in urban and suburban areas
- 🌾 **Rural & remote logistics** — reliable delivery where roads are poor or nonexistent
- 🍽️ **Perishables & food** — fast delivery without traffic delays
